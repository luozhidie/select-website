# 骆芷蝶智选 - 服装供应链 ToB 服务平台

> 让服装生意更容易 · 用个性化时尚改变每一个人生活

## 📖 项目介绍

骆芷蝶智选是一家专注于服装供应链服务的 ToB 平台，致力于让服装生意更容易。

我们通过严选供应商、专业选品指导、低门槛启动三大核心服务，帮助买手店、服装零售商、电商卖家解决找货难、选品难、启动难的痛点。

## 🚀 快速部署

### 方式一：Vercel 部署（推荐）

1. 访问 [vercel.com](https://vercel.com)
2. 注册/登录账号
3. 点击「Add New Project」
4. 选择「Import Git Repository」或拖拽上传文件
5. 点击「Deploy」

### 方式二：Netlify 部署

1. 访问 [netlify.com](https://netlify.com)
2. 注册/登录账号
3. 拖拽 `website` 文件夹到部署区域
4. 完成部署

### 方式三：GitHub Pages

1. 创建 GitHub 仓库
2. 上传 `website/index.html` 到根目录
3. Settings → Pages → 选择分支
4. 获得访问链接
